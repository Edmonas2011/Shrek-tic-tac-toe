# 🐸 Shrek Tic Tac Toe

A fun, Shrek-themed Tic Tac Toe game built with **HTML**, **CSS**, and **JavaScript**.  
Enjoy the classic Tic Tac Toe game with a unique twist inspired by the lovable Shrek character!
